# PRD: finsight

## 목표
카드 명세서 CSV를 올리면 Claude가 소비 패턴을 분석해 인사이트로 보여주는 SaaS.

## 사용자
자기 카드 소비를 빠르게 파악하고 싶은 개인. 엑셀로 직접 분석하긴 번거로운 사람.

## 핵심 기능
1. CSV(카드 명세서) 업로드 → 파싱·검증 → Storage + DB 저장
2. Claude 분석 → 인사이트(요약 + 카테고리별 표) 대시보드 표시
3. Polar 구독: Free(월 5건·기본 분석) / Pro(무제한·심층 분석)

## MVP 제외 사항
- 보상 트랜잭션·고아 파일 정리 잡 (실패 시 단순 에러)
- Claude 재시도·백오프·회로차단 (1회 호출, 실패 시 503)
- 한도 경계 동시성 원자화 (단순 카운트 조회)
- PII 마스킹 추가 레이어 (Supabase 기본 암호화 + RLS로 충분)
- 다중 결제수단·연간 플랜·환불 UI·쿠폰 (Polar 호스팅 페이지 위임)
- 소셜 로그인 다종·MFA (이메일/비번 1종)
- PDF/공유·차트 라이브러리 과다 (텍스트 + 간단 표)
- 관리자 대시보드·팀/조직·역할

## 디자인
- 다크모드 고정, 도구(대시보드)처럼. 마케팅 슬롭 금지.
- 무채색(neutral) + 포인트 색 1가지. 상세는 UI_GUIDE.md.
